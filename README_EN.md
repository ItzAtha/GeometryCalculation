# Geometry Volume Calculation Application

## 🧩 Application Description

This application is a mobile application designed to help users in calculating the volume of various geometrics such as cubes, blocks, tubes, cones, 
and spheres. The app is built using Flutter, a framework that allows cross-platform app development with a single codebase. The app has an interface 
that Simple and easy to use, making it suitable for students or anyone who wants to learn about geometrics volume calculation. Users can choose the 
type of geometrics that want to calculate, enter the required dimensions, and the application will provide the volume calculation results instantly.

## 🎯 Key Features

- **Volume Calculation**: Users can calculate the volume of various geometry shapes such as cubes, blocks, tubes, cones, and spheres.
- **Simple Interface**: Intuitive and easy-to-use design, suitable for all groups, especially students.
- **Cross-Platform**: Built with Flutter, the app can run on various platforms such as Android and iOS without the need to change the base code.
- **Responsive**: The app is designed to work well on a wide range of screen sizes, both on smartphones and tablets.

## 🛠️ Technology Used

- **Flutter**: A framework for building mobile apps developed by Google.
- **Dart**: The programming language used to develop Flutter applications.
- **Android Studio**: The IDE used to develop the Flutter application.
- **Git & GitHub**: For version control and code collaboration.
