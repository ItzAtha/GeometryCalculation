# Aplikasi Perhitungan Volume Bangun Ruang

## 🧩 Deskripsi Aplikasi

Aplikasi ini adalah sebuah aplikasi mobile yang dirancang untuk membantu pengguna dalam menghitung volume berbagai bangun ruang seperti kubus, balok, tabung, kerucut, dan bola. 
Aplikasi ini dibuat menggunakan Flutter, sebuah framework yang memungkinkan pengembangan aplikasi lintas platform dengan satu basis kode. Aplikasi ini memiliki antarmuka yang 
sederhana dan mudah digunakan, sehingga cocok untuk pelajar atau siapa saja yang ingin belajar tentang perhitungan volume bangun ruang. Pengguna dapat memilih jenis bangun ruang 
yang ingin dihitung, memasukkan dimensi yang diperlukan, dan aplikasi akan memberikan hasil perhitungan volume secara instan.

## 🎯 Fitur Utama

- **Perhitungan Volume**: Pengguna dapat menghitung volume berbagai bangun ruang seperti kubus, balok, tabung, kerucut, dan bola.
- **Antarmuka Sederhana**: Desain yang intuitif dan mudah digunakan, cocok untuk semua kalangan, terutama pelajar.
- **Lintas Platform**: Dibangun dengan Flutter, aplikasi ini dapat dijalankan di berbagai platform seperti Android dan iOS tanpa perlu mengubah kode dasar.
- **Responsif**: Aplikasi ini dirancang untuk bekerja dengan baik pada berbagai ukuran layar, baik di smartphone maupun tablet.

## 🛠️ Teknologi yang Digunakan

- **Flutter**: Framework untuk membangun aplikasi mobile yang dikembangkan oleh Google.
- **Dart**: Bahasa pemrograman yang digunakan untuk mengembangkan aplikasi Flutter.
- **Android Studio**: IDE yang digunakan untuk mengembangkan aplikasi Flutter.
- **Git & GitHub**: Untuk version control dan kolaborasi kode.
