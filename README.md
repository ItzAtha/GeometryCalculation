# Aplikasi Perhitungan Volume Bangun Ruang

## 🧩 Deskripsi Aplikasi

Pada tugas pertama ini, saya diminta untuk membuat sebuah aplikasi mobile untuk melakukan perhitungan volume pada bangun ruang.
Aplikasi ini dibuat menggunakan **Framework Flutter** dengan **Dart** sebagai basis bahasa nya. Aplikasi ini memiliki beberapa fitur utama, yaitu:
- **Input Data**: Pengguna dapat memasukkan data yang diperlukan untuk menghitung volume bangun ruang, seperti panjang, lebar, dan tinggi.
- **Perhitungan Volume**: Aplikasi akan melakukan perhitungan volume berdasarkan data yang dimasukkan oleh pengguna dan menampilkan hasilnya.
- **Tampilan User-Friendly**: Aplikasi ini dirancang dengan antar muka yang sederhana dan mudah digunakan, sehingga pengguna dapat dengan mudah memahami cara menggunakan aplikasi ini.

Di dalam aplikasi ini, saya juga menambahkan beberapa fitur tambahan seperti validasi input untuk memastikan bahwa data yang dimasukkan oleh pengguna benar dan sesuai dengan format yang diharapkan.

## 🛠️ Teknologi yang Digunakan

- **Flutter**: Framework untuk membangun aplikasi mobile yang dikembangkan oleh Google.
- **Dart**: Bahasa pemrograman yang digunakan untuk mengembangkan aplikasi Flutter.
- **Android Studio**: IDE yang digunakan untuk mengembangkan aplikasi Flutter.
- **Git & GitHub**: Untuk version control dan kolaborasi kode.
